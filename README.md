# **The Lag**

## Installation Première utilisation :

1. Download docker / install docker

## Lancement du projet :

1. `docker-compose up -d` à la racine
2. Faites vous plaisir

## Arret du projet :

1. `docker-compose down`
2. Prenez vous une petite pause
